Demo of "periodic-table"

Create a visible periodic table: createList(document.getElementById("elements"))
The class list of each element contains "periodic-table".

Get the information of the element: getElement("")
You can put any of these in the input box above: "8" "o" "oxygen" "氧" "15.999"

Get the information of the compound: getCompound("")
You can put any of these in the input box above: "carbon dioxide" "CO2" "c o2" "二氧化碳"
